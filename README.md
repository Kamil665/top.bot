# Telegram Bot — Регистрация пользователя

Простой Telegram-бот на Python для регистрации пользователя с использованием библиотеки `pyTelegramBotAPI`.

---

# Возможности

Бот умеет:

- запускать регистрацию командой `/reg`
- спрашивать:
  - имя
  - фамилию
  - возраст
- подтверждать введённые данные кнопками:
  - «Да»
  - «Нет»
- перезапускать регистрацию при отказе

---

# Используемые технологии

- Python 3
- telebot (`pyTelegramBotAPI`)
- InlineKeyboardMarkup
